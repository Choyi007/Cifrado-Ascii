# Cifrado-Ascii
Hice este cifrado de ASCII para un programa que hize, no sé cómo resolví la creación del descifrado, llevo 6 meses programando en Python así que disculpen la desorganizacon y todo eso xd

Codigo Creado por Cirsev, para inicializarlo ejecuten mediante Core.py, tambien pueden convertirlo a .exe si quieren, este codigo te permite almacenar contraseñas o datos importantes como CURP, INE, ETC...

Archivos:
  - Cipher.py
    este es el modulo de cifrado, cuando creas una llave, osea cuando añades un dato, pasa por este cifrado, hubiese usado alguna       
    libreria pero crei que aprenderia mas si hacia el cifrado desde 0
    
  - Core.py
    el nucleo del sistema, contiene todos los comandos y conecta los modulos.
    
  - JsonModule.py
    carga, guarda y crea de forma segura los archivos .json que contendran los datos

  - README.md
    esto que estas leyendo xd

  - TimeStapModule.py
    el archivo de logs tenia errores al poner la tiempo, asi que decidi separarlo del codigo principal y hacerlo de forma segura

  - User Interactions.py
    las interaciones con el usuario, verifica que cada accion que realize el Usuario sea correcta, preguntandole y evitando errores
    con los Inputs
